使用hexo + yilia搭建blog

笔记

配置blog

1.安装node.js node.js
2.检查版本

node -v npm -v

3.利用nmp安装cnmp

npm install -g cnpm --registry=https://registry.npm.taobao.org
cnpm //检查是否安装好
cnpm -v

4.安装hexo

cnpm install -g hexo-cli
hexo -v

5.初始化

//on windows
chdir //查看当前目录
mkdir blog / md blog
cd blog
hexo init
dir
hexo -s //可以在localhost:4000 查看
hexo -n "文件名"
chdir
cd source/_posts/
dir

6.使用vs code 打开 要编辑的文件

cd ../.. //回退两级目录
hexo clean //清理
hexo g //生成
hexo s //开始

7.部署到GitHub

新建仓库 用户名.github.io
在blog目录下安装一个git插件

cnpm install --save hexo-deployer-git

修改配置文件 _config.yml
在文件最底部 Deployment

type: git
repo: 仓库地址
branch: master

部署
hexo -d

输入账号

输入密码

更换blog主题

1.克隆主题

git clone https://github.com/litten/hexo-theme-yilia.git themes/yilia
dir

2.修改_config.yml

把theme：修改为yilia

hexo clean
hexo g
hexo s
ctl + c
hexo d

完工惹 以后就可以愉快的写blog了(划水(误 前面划掉(之后会自己定制一套个人博客(先开一个坑 不知道什么时候填
