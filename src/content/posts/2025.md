2025 年其实蛮特殊的：这是我参加工作后 gap 的完整一年。回首这一年，好像没有太多大事发生，相对平静。我时常不知道今天是星期几，也不用焦虑今天几点睡、明天几点起——有一种时代的洪流滚滚向前、而我在岸边旁观的错觉。

我不认为这是长久之计，但我也同样不觉得人生暂停休整一下是什么天大的事情。

## 感情

去年谈了一段恋爱。今年 3 月末，我被前任家暴，弄了一身伤口，连夜离开。之后去朋友家住了几天，可以说极为狼狈。

那天我也果断分手了。群友们还调侃说：这么多年还是第一次见男人被家暴。

我自己花了一个月把情绪慢慢平复下来，之后搬家到余杭，因为这里离朋友们更近一些。

## 学习

作为一个程序员我认为自己基础知识还远远不够扎实,编码能力还很弱。

今年我重新把工作时半途而废、想要学习的课程捡起来又学了一遍，主要是下面两门：

- NJU 软件分析课程：学习并完成所有实验（做实验的过程巨痛苦，花了很多的时间找一些 case 为什么过不了 尤其是 OJ 上的 hidden case）。
- NJU JYY OS 课程：完成学习（但还没有做实验，所以严格来说不算“完成”）。

整体非常有意思：我了解到了很多以前不知道的知识。更有趣的是，它们让我能从不同视角重新认识这台我每天都在使用的计算机。

今年完整读完的书不多，应该只有两本：

- 《稀缺》
- 《太白金星有点烦》

另外还有一些只读了一部分：

- 《爱欲与哀矜》
- 《持续交付 2.0》
- 《软件设计哲学》

## Code

虽然是在 gap 但是不写代码是不可能忍得住的。

给 naive-ui 加了一些我自己想做的 feature 和一些改进：加了一些组件，修了一些 bug，也清了一些债务。

提的 PR 不算多，有的合了，有的还没 review；有点子了就会写写。

给 css-render 做了 VS Code 插件，学习了一点插件开发的知识，不过还有一些 bug 没来得及修。

在用其他开源项目时零零碎碎遇到了一些问题，力所能及的范围内也提了一点 PR。

年中的时候朋友说需要帮忙用 Astro 写点东西，弄完之后我顺手学习了一下 Astro，又用 Astro vibe 了这个 blog 站点，放了一些平时写文章上去。

### 行业观察

今年行业最显著的关键词就是 **AI 与 Agent**。

一边是基础模型的竞速：年初 DeepSeek V3 / R1 掀起全民讨论；海外有 OpenAI、Anthropic、Google，国内也涌现出 DeepSeek、Kimi、MiniMax、Qwen、豆包/Seed 等。

另一边是应用侧的加速落地：写代码的工具链（Cursor、Claude Code、Codex 等）快速迭代；图像生成也持续升温，阶段性最出圈的当属 Nano Banana。

除头部玩家外，还有大量创业团队与项目入场，互联网公司也纷纷推出自己的 Agent 产品。体感上，这股浪潮确实“浩浩荡荡”——既有机会与风光，也伴随更高的不确定性。

对身边还在上班的朋友而言，这一年更像是主动/被动地卷进 AI：迭代快、交付密度高，加班几乎成了默认配置。与此同时，招聘端的体感也很明显：传统岗位缩减，AI 相关需求仍在增长。

我选择积极学习各种概念与工具，拥抱 AI。但越是“疯狂使用”，越容易把判断力外包给它，久而久之思维会变钝。我们需要刻意保持思考与训练：把 AI 当作一个好的协作伙伴、补齐关键上下文与约束、做取舍并承担结果，成为 AI 的好 leader。

“AI Agent 的潮流可以说是浩浩荡荡，每个人都身处洪流之中。其间，有些人凭着自身的努力或者说是幸运站在了潮头之上，这潮头之上是风光无限，诱惑无限，也风险无限就看你如何把握了，看未来远不如看过去要来的清楚，激昂和困惑交织在每个人的心头”

## 生活

因为没有上班，我得以有空自己做饭。时间不紧迫的时候我还挺喜欢做饭的；不想做的时候就点外卖。周末和朋友聚餐，有时候做多了，我就把附近的哥们儿叫来家里一起吃饭聊天，或者给他们打包一点让他们过来拿。

今年染上的新爱好是唱 K。以前去 KTV 因为有不熟悉的人，我一般就是坐牢；现在有稳定的朋友们，今年一年唱的 K 比我过去 20 多年加起来都多，很开心也很解压。

### 手术

9 月我住院做了手术，把 2 年前因为骑行骨折植入的两块内固定物取出，修养了一段时间。

在这之前，手术是时不时会让我焦虑的事情，也是为数不多会引起我焦虑情绪的事。我也因此在网上看了各种说法，担心它的风险。好在手术顺利，手术之后心情也好了一些。

### 游戏

作为一个以前很少玩儿游戏的人，今年在 Steam 大促买了十几个游戏。当然目前我只玩儿了其中几个，但也久违地感受到了电子游戏带来的快乐。

Steam 游戏玩儿了：双人成行、土豆兄弟、逃离塔科夫、风来之国、丝之歌、情感反诈模拟器等。

手游玩儿了：三角洲、不存在的谎言。

## 碎碎念

近两年我买了很多以前舍不得给自己买的东西：电脑、相机、显示器、椅子。以前有点不敢对自己好，现在觉得人生苦短，应该多体验。

很多东西还是应该早点买：早买早享受，没必要没苦硬吃。

对比如，显示器、椅子、电脑这种生产力工具还是要早点买，也免得身体受罪，买完之后幸福感提升了非常多。

一年的时间过得真的太快了,回头看去，这一年总体来说这也是收到了很多爱、也给出了很多爱的一年。虽然有的爱并没有被好好珍惜，但希望我一直能有勇气给出爱。

我对 2026 并没有太多具体的规划 大抵是会好好找个班上赚点钱满足我生长的物欲 (朋友说好想听我回去上班后的狗叫🐶),也期望能更深入的学习计算机,多出去看看世界，了解世界，拍到更多满意的照片

相对自己和读到这里的各位朋友们说一句话：

很多事情远没有眼前看的那么重要，身体第一，记得好好热爱生活，尽兴的活着，好好体验这个世界吧！！！
